Fonctionnalités

Tout ce qu'il vous faut pour gérer la course

Conçu pour les équipes logistiques, simple à utiliser en conditions réelles.

🚴
3 vélos simultanés
Grand vélo, petit vélo et folklo — chacun géré indépendamment avec son propre chrono et sa file de riders.
⏱
Chronomètres temps réel
Suivi de chaque tour en direct. Alertes automatiques si un rider reste trop longtemps sur la piste.
🔄
Gestion des transitions
File d'attente par vélo, changement de rider en un clic, timer de transition intégré.
📊
Analytics complets
Vitesse moyenne, tours totaux, distance par vélo et par rider — visualisation instantanée des performances.
💾
Export Excel & PDF
Export complet des données de course en un clic. Backup JSON pour restaurer une course à tout moment.
🌙
Mode jour / nuit
Thème adapté selon les conditions — mode nuit pour la piste, mode jour pour le briefing en salle.
